**🐧 Linux Troubleshooting Runbook**  
A searchable **L1/L2 support knowledge base** built with vanilla HTML/CSS/JS and deployed on GitHub Pages. Covers 20 real-world Linux support scenarios with step-by-step diagnosis, commands, and resolution.  
**Live site →** https://swapnajitsarkar.github.io/linux-troubleshooting-runbook/

**🔍 Features**  
- **Sidebar search** — filter all scenarios in real time by keyword (e.g., permission, cron, OOM)  
- **Expandable cards** — click any scenario to expand diagnosis steps and commands  
- **Categorized sections** — filesystem, processes, networking, services  
- **Bash one-liner reference table** — copy-paste ready, no explanation needed  
- **Common mistakes appendix** — quick checklist before deep investigation  
- **Dark terminal theme** — easy to read on a second monitor during an incident  
- **No dependencies** — pure HTML/CSS/JS, no Node, no Jekyll, no build step
